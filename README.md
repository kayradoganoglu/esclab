# ESClab v0.2.2 README
## Dependencies:
  **PyQt6** >= *6.7.1*<br>
  **PyQt6-WebEngine** >= *6.7.0*<br>
  **numpy** >= *2.0.2*<br>
  **pandas** >= *2.2.2*<br>
  **plotly** >= *5.24.0*<br>
  **matplotlib** >= *3.4.3*<br>
*Please install packages using* **pip**.<br>
Use command:<br>
**pip install numpy pandas matplotlib plotly pyqt6 pyqt6-webengine**<br>
*After package installation* run **main.py**
